# 通用 Prompt — Amazon 賣家工具集產生器

> 複製以下 prompt，將 `【】` 內的內容替換成目標站點資訊，貼到 Kiro 新 session 即可產生對應站點的工具。

---

## PROMPT 開始 ↓ 複製這裡以下

---

請幫我製作一個「Amazon【站點名稱】賣家工具集」的互動式 Web 應用（React + TypeScript + Tailwind CSS），整合以下三個模組在同一個單頁應用中，用分頁切換：

### 技術要求
- React + TypeScript + Tailwind CSS + Vite
- 所有資料用 JSON/TypeScript 結構管理（方便日後更新）
- 純前端，不需後端
- 響應式設計（手機可用）
- 中文介面
- 可部署為靜態網站（Netlify / Vercel）

### 目標站點國家
【請列出國家，格式如下】
- 【國家1中文名】（【國家代碼】）— 貨幣：【貨幣】，VAT稅率：【稅率】%
- 【國家2中文名】（【國家代碼】）— 貨幣：【貨幣】，VAT稅率：【稅率】%
- （依此類推...）

---

### 模組一：開站全流程引導（Wizard）

以 step-by-step 形式引導賣家完成開站準備，包含以下步驟：

1. **前置作業** — 公司資料、銀行帳戶、信用卡、Email、電話
2. **帳號註冊** — Seller Central 註冊流程、視訊驗證
3. **VAT 稅務** — 各國 VAT 稅號註冊、定期申報、稅務代表（如需要）
4. **產品安規** — 【該區域的產品安全標誌，如 CE/UKCA/PSE/KC 等】、GPSR 或當地產品安全法規、授權代理人
5. **環保法規** — EPR、包裝法、電子廢棄物回收（WEEE 或當地等效法規）、電池法規等
6. **通關與稅號** — EORI 或當地進口稅號、海關要求
7. **FBA 物流** — FBA 計畫選擇、首批入倉準備
8. **帳務與金流** — 收款設定、發票要求、帳務處理

每個步驟的每個項目需包含：
- 標題和說明
- 適用國家標籤
- 是否必要（必要/視品類而定）
- 所需文件清單（含範例欄位）
- 預估時程
- 預估費用
- 難易度（1-5 星）
- 實用提示
- 資料來源標註

功能：
- 互動式 checklist，可勾選已完成項目
- 整體進度條
- 各步驟完成狀態一目了然

---

### 模組二：產品安規查詢工具

賣家選擇商品品類 + 國家，立即顯示：

1. **安規認證要求** — 該品類在各國必須取得的認證，含時程、費用、難易度
2. **Amazon 佣金費率（Referral Fee）** — 該品類在各國的佣金費率（請搜尋最新的 Amazon 【站點名稱】費率資料）
3. **環保合規** — EPR、包裝法、電子廢棄物等適用要求
4. **授權代理人** — 當地法規要求的授權代理人/負責人

請包含至少以下品類（可依該站點特性調整）：
- 消費電子
- 玩具
- 服飾與配件
- 家居用品
- 美妝保養
- 食品雜貨
- 寵物用品
- 運動戶外
- 汽車零配件
- 保健食品/維他命

---

### 模組三：開帳號前置作業清單

賣家勾選「要註冊的國家」和「FBA 入倉國家」後，自動產出：

1. 所有需要辦理的項目清單，依類別分組（稅務/產品安全/環保/註冊通關）
2. 每個項目顯示：
   - 申請時程
   - 費用估算
   - 難易度（星級）
   - 適用國家
   - 必要 vs 建議
   - 所需文件
   - 實用提示
3. 進度追蹤（可勾選已完成）
4. 總覽卡片：待辦數量、已完成、完成率、預估最低費用

---

### 設計原則
- 目的是降低人力說明成本，讓賣家自助式快速釐清所有事項
- 易懂、直覺、資訊分區清楚
- 用顏色/圖示區分：必要 vs 建議、難易度、完成狀態
- 費用和時程一目了然
- 每個資訊都標註來源

### 資料來源
- 請從外部搜尋可信賴的官方資料（Amazon Seller Central、各國政府官方網站、官方法規）
- 所有引用資料請標註來源
- 費用和時程為估算值，需註明僅供參考

## 使用範例

### 範例 1：Amazon 日本站
將【】替換為：
- 站點名稱 → 日本
- 國家 → 日本（JP）— 貨幣：JPY，消費稅：10%
- 產品安全標誌 → PSE、PSC、食品衛生法、電波法（技適マーク）
- 環保法規 → 容器包裝リサイクル法、家電リサイクル法、小型家電リサイクル法

### 範例 2：Amazon 北美站
將【】替換為：
- 站點名稱 → 北美
- 國家 →
  - 美國（US）— 貨幣：USD，Sales Tax：各州不同
  - 加拿大（CA）— 貨幣：CAD，GST：5% + 各省 PST
  - 墨西哥（MX）— 貨幣：MXN，IVA：16%
- 產品安全標誌 → FCC、UL、FDA、CPSC、NOM
- 環保法規 → EPA、Prop 65（加州）、各州包裝法

### 範例 3：Amazon 東南亞站
將【】替換為：
- 站點名稱 → 東南亞
- 國家 →
  - 新加坡（SG）— 貨幣：SGD，GST：9%
  - 澳洲（AU）— 貨幣：AUD，GST：10%
- 產品安全標誌 → SAA、RCM、ACMA
- 環保法規 → 澳洲包裝公約、NTCRS
